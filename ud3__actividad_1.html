UF2217. UD3. Actividad 1. Caso práctico: CONSTRUCCIÓN DTD’S INTERNAS Y EXTERNAS.


Enunciado (Descripción)

Dado un documento XML, construir su DTD, utilizar la DTD como interna y como externa, en ambos casos comprobar la validez

Descarga el documento adjunto: UF2217_UD3_Actividad 1_Caso práctico_AA.rar

contacto.xml


Preguntas / Actividades a realizar 

Partiendo del documento de apoyo, construir DTD que describa la estructura del documento XML de apoyo. Incorporar esta DTD, como una DTD interna al documento XML de partida, verificar que es un documento válido con ‘XML Copy Editor’. Con la herramienta ‘XML Copy Editor’, guardar en el sistema de archivos, la DTD construida y hacer referencia a ella desde el XML de partida. Verificar que este XML con la DTD externa es un documento válido.
