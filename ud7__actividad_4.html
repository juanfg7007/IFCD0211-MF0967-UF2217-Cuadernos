UF2217. UD7. Actividad 4. Caso práctico grupal: Instalación de CURL


Enunciado (Descripción)

Esta actividad se realizará de manera grupal, para ello tu tutor-formador creará distintos grupos de trabajo y te indicará a qué grupo perteneces, los miembros que lo forman y el espacio de la plataforma que se habilitará para que podáis desarrollar la actividad.

Instalar CURL para enviar petición al servidor remoto http://www.example.com/test.php mediante el protocolo POST.

CURL es una herramienta para usar en un intérprete de comandos para transferir archivos con sintaxis URL, soporta FTP, FTPS, HTTP, HTTPS, TFTP, SCP, SFTP, Telnet, DICT, FILE y LDAP. CURL soporta certificados HTTPS, HTTP POST, HTTP PUT.

De esta forma reforzaremos el concepto XML-RPC.


Preguntas / Actividades a realizar 

Conseguir instalación de Curl, recordar que es multiplataforma. En caso de disponer de entorno Unix, probablemente esté instalado, si esa estuviera deshabilitada, habría que ejecutar la instrucción super usuario: sudo apt-get install curl

También será necesario revisar si la instalación PHP tiene la librería Curl habilitada, en caso contrario necesitaremos otra instrucción desde la línea comandos:

sudo apt-get install php5-curl

Realizar petición Post al servidor http://www.example.com/test.php

Una vez que todos los grupos de trabajo realicen la tarea encomendada, el tutor-formador recopilará toda la información aportada por los distintos grupos de trabajo, creando un único documento donde se recogerán todos los aspectos trabajados en esta actividad, y lo subirá a la plataforma para que todos los alumnos puedan hacer uso del mismo.

  • -X POST: especifica que se enviará una solicitud POST al servidor.

  • -d "dato1=valor1&dato2=valor2": especifica los datos que se enviarán en el cuerpo de la solicitud POST.

  • http://www.example.com/test.php: especifica la URL del servidor y el script PHP al que enviar la solicitud POST.
